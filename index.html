How I Use AI in Teaching

I do use AI to...

  • Review and clarify my lecture notes
  • Generate ideas for activities and assignments
  • Create exam questions and distractors
  • Get advice on teaching challenges

I don't use AI to...

  • Write my lecture content
  • Write code I present in class
  • Grade student work

The Process: Building a Tool

  1. Start with a request: Ask for a single-page web app (HTML, CSS, JavaScript) that demonstrates your concept
  2. Provide context: Share your lecture notes, analogies, or examples of tools you like
  3. Iterate: Have a conversation! Ask for fixes, new features, and refinements (expect 20-40 rounds)
  4. Test and refine: Use the tool yourself, notice what's confusing, ask for improvements
  5. Deploy: Copy the HTML file to a web server, LMS, or GitHub Pages
Key insight: You don't need to read or write any code. Describe what you see, describe what you want, and let the AI handle the implementation.

Step-by-Step Workflow

  1. Open an AI assistant (claude.ai, chat.openai.com, gemini.google.com) in a browser tab
  2. Describe what you want: "Create a single-page web app using HTML, CSS, and JavaScript that demonstrates [your concept]..."
  3. The AI responds with code, get it onto your computer using one of these options:
    • Download: Some assistants have a Download button, click it and save as myDemo.html
    • Copy/Paste: Click Copy, open a text editor (TextEdit, Notepad), paste, and save as myDemo.html
  4. In your file explorer, double-click the file and it will open in your browser
  5. Try it out, notice what's wrong or missing
  6. Go back to the AI chat and describe the problem: "The button doesn't work" or "Can you make the text bigger?"
  7. Download the new code (or copy/paste over the old file), save, refresh the browser
  8. Repeat steps 5-7 until you're happy (expect 20-40 iterations)
